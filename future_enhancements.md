# 6. Future Enhancements

## Planned Features
- **Quick Settings Tile**: Add a native Android Quick Settings tile so users can trigger the power menu directly from their system notification drawer without opening the application.
- **Home Screen Widget**: Design an ultra-minimal home screen launcher widget for instant power menu activation.
- **Lock Screen/Notification Shortcut**: Optional persistent, low-overhead notification bar utility for rapid access.

## Technical Improvements
- **Automated UI Testing**: Implement local screenshot tests utilizing Roborazzi and Robolectric to verify UI alignment across layout size classes.
- **Edge Case API Checks**: Refine layout compatibility for extremely small aspect-ratio screens.
